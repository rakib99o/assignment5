Question-1: What are the different ways to select an element in the DOM?

Elements in the DOM can be selected using methods like `getElementById()`, `getElementsByClassName()`, `getElementsByTagName()`, `querySelector()`, and `querySelectorAll()`.

Question-2: What is the difference between innerHTML, innerText, and textContent?

`innerHTML` includes HTML tags and content, `innerText` retrieves visible text while considering styling (like hidden elements), and `textContent` retrieves all text, regardless of styling or visibility.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a technique where a single event listener is attached to a parent element, instead of individual child elements, and it relies on event bubbling to handle events on dynamically added or existing child elements.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event triggered on a child element propagates upward through its parent elements in the DOM until it reaches the root.
